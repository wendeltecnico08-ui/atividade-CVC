Instruções
Exemplo de comando rápido
git clone https://github.com/SEU_USUARIO/aula-github-seu-nome.git

Atividade:

  1. Clonar o repositório do professor.
  2. Editar o arquivo index.html
  3. Fazer commit e push da sua branch.
  4. Ativar Github Actions
  5. Visualizar o site publicado.
  6. O colega da atividade deve fazer um pull request no seu repositório adicionando o nome dele
